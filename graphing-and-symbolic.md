## Introduction

As well as the standard scientific calculator, [scientificcalculators.net](https://www.scientificcalculators.net) includes two powerful advanced tools:

- **Graphing Calculator** — plot and compare mathematical functions on an interactive 2D graph
- **Symbolic Computation** — simplify and expand algebraic expressions with variables, with step-by-step working

Both are accessible from the tab bar at the top of the page. This guide covers how to use both tools effectively.

## Part 1: The Graphing Calculator

### How to Access

From the homepage at [scientificcalculators.net](https://www.scientificcalculators.net), click the **graph icon** tab (the middle tab in the tab bar at the top of the calculator). You can also navigate directly via the Graphing Calculator menu link.

---

### Entering Functions

1. Type your function in the input field — use `x` as the variable
2. Click the orange **+** button or press **Enter** to plot it
3. The function appears in the list below the input field, with a coloured dot showing its line colour on the graph

**Example expressions to try:**
- `sin(x)` — sine wave
- `x^2` — parabola
- `x^2 / 10` — scaled parabola
- `sin(x) + cos(x)` — combined trig
- `sqrt(x)` — square root curve
- `1/x` — hyperbola
- `log(x)` — logarithmic curve
- `2*x + 1` — straight line

### Supported Functions

| Function | Syntax |
|----------|--------|
| Sine | `sin(x)` |
| Cosine | `cos(x)` |
| Tangent | `tan(x)` |
| Log base 10 | `log(x)` |
| Natural log | `ln(x)` |
| Square root | `sqrt(x)` |
| Absolute value | `abs(x)` |
| Exponential | `exp(x)` |
| Power | `pow(x, y)` or `x^y` |

---

### Managing Multiple Functions

You can plot as many functions as you like simultaneously — useful for comparing curves or finding intersections.

Each function in the list has two controls:
- **Eye icon** — toggle visibility (hide/show the line without deleting the function)
- **Trash icon** — delete the function permanently

To edit a function, simply click into its input box and retype — the graph updates in real time.

---

### Graph Settings

Below the function list, the **Graph Settings** panel lets you customise the view:

**X-Axis Range** — set the minimum and maximum x values (default: −10 to 10)

**Y-Axis Range** — set the minimum and maximum y values (default: −10 to 10)

**Zoom Level** — use the slider to zoom from 10% to 500%

**Reset View** — returns all settings to the default −10 to 10 range for both axes

**Clear All** — removes all plotted functions and resets the graph

---

### Interacting with the Graph

- **Click and drag** on the graph to pan around
- **Zoom controls** on the graph itself (+ and − buttons) let you zoom in and out
- **Camera icon** — save a screenshot of the current graph
- **Home icon** — return to the default view

---

### Tips for Best Results

- Always use `x` as your variable — other letters won't plot
- Use parentheses to avoid ambiguity: `sin(x)` not `sinx`
- If a graph doesn't appear, check for syntax errors or unmatched brackets
- For functions with asymptotes (like `1/x` or `tan(x)`), the graph may show a vertical line at the discontinuity — this is normal display behaviour
- Adjust the axis ranges if your function's interesting behaviour is outside the default −10 to 10 window

## Part 2: Symbolic Computation

### How to Access

Symbolic computation is built into the main **Scientific Calculator** tab. Scroll down below the keypad on the calculator page to find the **Symbolic Computation** section.

---

### What It Does

The symbolic computation engine lets you enter algebraic expressions containing variables (x, y, z) and simplify them — showing the step-by-step working.

**Supported operations:**
- Combining like terms (e.g. `2x + 3x` → `5x`)
- Adding and subtracting constants (e.g. `5 + 3` → `8`)
- Basic bracket expansion (e.g. `(x + 2)(x − 3)` → `x² − x − 6`)
- Simplifying mixed expressions (e.g. `3x + 5 − x + 2` → `2x + 7`)
- Variables: x, y, z

---

### How to Use It

1. Click the **"Enter e"** input box in the Symbolic Computation section
2. Type your algebraic expression
3. Click the orange **Simplify** button
4. The simplified result appears, along with the step-by-step working

Click **Clear** to reset the input and start a new expression.

---

### Example Expressions

Click any of the pre-loaded example buttons to see them in action:

| Input | Result |
|-------|--------|
| `2x + 3x` | `5x` |
| `x^2 + 2x + 1` | `(x+1)²` |
| `(x + 2)(x − 3)` | `x² − x − 6` |
| `3x + 5 − x + 2` | `2x + 7` |

---

### Viewing Steps

After simplifying, the calculator shows the step-by-step process — useful for checking your working or learning how the simplification was carried out.

---

### What Symbolic Computation Is Not

The symbolic computation tool handles simplification and basic expansion. It is not a full computer algebra system — it won't solve differential equations or perform calculus. For those tasks, use the graphing calculator to visualise functions.

## Practice Problems

Work through these using the free tools at [scientificcalculators.net](https://www.scientificcalculators.net):

**Graphing:**
1. Plot `sin(x)` and `cos(x)` on the same graph — where do they intersect?
2. Plot `x^2` and `x^3` — how do they differ for x > 1 vs x < 1?
3. Set X range to 0–20 and plot `log(x)` — how quickly does it grow?

**Symbolic:**
1. Simplify `4x + 7x − 2x` — Answer: `9x`
2. Expand `(x + 5)(x − 5)` — Answer: `x² − 25`
3. Simplify `6y + 3 + 2y − 1` — Answer: `8y + 2`
